# fxkqmc
qmc decoder for python

## Usage

```
usage: qmc_decoder.py [-h] -i INPUT [-o OUTPUT] [-f]

QMC decoder

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        input dir
  -o OUTPUT, --output OUTPUT
                        output dir, default will create /output dir under
                        input path
  -f, --force           overwirte output dir
```
